# Asimov — Landing Pages

**Quais ferramentas de IA você usou?** Claude Design, Claude Code e GitHub Copilot.

**Em que etapas a IA ajudou mais?** Na `parte-1`, o maior ganho foi na velocidade de escrita do código a partir da exportação do arquivo `.fig`, restando apenas ajustes pontuais. Na `parte-2`, a IA acelerou tanto a codificação quanto a definição do design, gerando o layout a partir de prompts detalhados e estruturados.

**O que você ajustou manualmente e por quê?** Inseri `padding-bottom` no header para acomodar o comportamento _sticky_ e melhorar o visual; substituí os SVGs gerados pela IA pelos assets exportados do Figma que já havia incluído no contexto; reorganizei a estrutura de pastas para maior clareza; e apliquei melhorias de acessibilidade, ajustando o CSS para navegação por teclado e adotando tags HTML mais semânticas.

**Observação:** Na `parte-2`, substituí o nome da empresa por "Vomisa" (Asimov invertido) para evitar publicar um site com a identidade da Asimov sem autorização.
