# Asimov — Landing Pages

Duas landing pages estáticas (`parte-1/` e `parte-2/`) desenvolvidas em HTML, CSS e JS, publicadas na Vercel.

As ferramentas de IA utilizadas foram Claude Design, Claude Code e GitHub Copilot.

Na `parte-1`, o maior ganho foi na velocidade de escrita do código a partir da exportação do arquivo `.fig`, restando apenas ajustes pontuais. Na `parte-2`, a IA acelerou tanto a codificação quanto a definição do design, gerando o layout a partir de prompts detalhados e estruturados.

Entre os ajustes manuais, inseri `padding-bottom` no header para acomodar o comportamento *sticky* e melhorar o visual; substituí os SVGs gerados pela IA pelos assets exportados do Figma que já havia incluído no contexto; reorganizei a estrutura de pastas para maior clareza; e apliquei melhorias de acessibilidade, ajustando o CSS para navegação por teclado e adotando tags HTML mais semânticas.
